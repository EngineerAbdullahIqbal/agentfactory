## Usage rules

- Keep English terms that have no standard Hindi translation (e.g., API, URL, HTTP)
- Use the same translation for a term everywhere in the document
- Add new terms as they appear during translation
- Technical terms commonly used in English by Hindi-speaking developers should stay in English
- Use Devanagari script for all Hindi translations
- Use Hindi punctuation: पूर्ण विराम (।) instead of period where appropriate, but period (.) is also acceptable in technical docs
