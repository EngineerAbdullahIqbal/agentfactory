## Output Folder Structure

Final translated files go under `apps/learn-app/i18n/<locale>/`:

```
apps/learn-app/i18n/<locale>/
├── code.json                                    # Theme UI strings
├── docusaurus-plugin-content-docs/
│   ├── current.json                             # Sidebar category labels
│   └── current/                                 # Translated docs (mirrors docs/)
│       ├── thesis.md
│       ├── 01-General-Agents-Foundations/
│       │   └── 01-agent-factory-paradigm/
│       │       ├── 01-the-2025-inflection-point.md
│       │       └── ...
│       └── ...
└── docusaurus-theme-classic/
    ├── footer.json                              # Footer strings
    └── navbar.json                              # Navbar strings
```

## Path Resolution

| Source Type | Source Location | Output Location |
|-------------|---------------|-----------------|
| Lesson `.md`/`.mdx` | `apps/learn-app/docs/<path>/<file>.md` | `apps/learn-app/i18n/<locale>/docusaurus-plugin-content-docs/current/<path>/<file>.md` |
| Theme strings | (generated by Docusaurus) | `apps/learn-app/i18n/<locale>/code.json` |
| Sidebar labels | (generated by Docusaurus) | `apps/learn-app/i18n/<locale>/docusaurus-plugin-content-docs/current.json` |
| Navbar labels | (generated by Docusaurus) | `apps/learn-app/i18n/<locale>/docusaurus-theme-classic/navbar.json` |
| Footer labels | (generated by Docusaurus) | `apps/learn-app/i18n/<locale>/docusaurus-theme-classic/footer.json` |

## Working Directory Structure

Intermediate artifacts go in `apps/learn-app/translation-work/`:

```
apps/learn-app/translation-work/
├── manifest.json                  # State tracker
├── ur/                            # Per-locale working files
│   ├── thesis_ur_segments.json
│   ├── thesis_ur_translated.json
│   └── ...
├── fr/
└── es/
```

**Naming convention:**
- Segments: `<locale>/<filename-without-ext>_<locale>_segments.json`
- Translated: `<locale>/<filename-without-ext>_<locale>_translated.json`

**Example path chain:**
```
Source:     docs/01-General-Agents-Foundations/01-agent-factory-paradigm/01-the-2025-inflection-point.md
Segments:  translation-work/ur/01-the-2025-inflection-point_ur_segments.json
Translated: translation-work/ur/01-the-2025-inflection-point_ur_translated.json
Output:    i18n/ur/docusaurus-plugin-content-docs/current/01-General-Agents-Foundations/01-agent-factory-paradigm/01-the-2025-inflection-point.md
```

## Important Notes

- Keep original filename (Docusaurus uses folder-based i18n, NOT file-suffix like `file.ur.md`)
- Output valid Markdown/MDX only
- JSON files use 2-space indent with trailing newline
