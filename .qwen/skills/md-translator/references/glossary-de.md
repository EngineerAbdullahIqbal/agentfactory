## Usage rules

- Keep English terms that have no standard German translation (e.g., API, URL, HTTP, Token, Framework)
- Use the same translation for a term everywhere in the document
- Add new terms as they appear during translation
- German compound nouns are written as one word (e.g., Laufzeitumgebung, not Laufzeit Umgebung)
- Capitalize all nouns in German text
