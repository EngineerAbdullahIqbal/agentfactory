# Claude Cowork

Claude Cowork brings agentic AI from the terminal to a desktop application that non-developers can use. This section covers getting started, practical workflows, browser integration with Claude in Chrome, the Plugin ecosystem (bundling connectors, skills, slash commands, and sub-agents into portable workflow packages), and the safety model that governs what Cowork can and cannot do.

**Lessons 24-30** give you hands-on experience with Claude's desktop agent, its Plugin ecosystem with 50+ connectors, cross-app document orchestration, and built-in skills.
